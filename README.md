# Joymouse
A mouse replacement using an Arduino Micro, joystick, and rotary encoder.

Analog joystick:
  Power         5V
  Ground
                x-axis        pin A0
                y-axis        pin A1
                left click    pin D3

Rotary encoder:
  scroll up     pin D6
  scroll down   pin D7
  right click   pin D4
