# Joymouse
A mouse replacement using an Arduino Micro, joystick, and rotary encoder.

Analog joystick:              <br />

Power         5V              <br />
Ground                        <br />
                x-axis        pin A0      <br />
                y-axis        pin A1      <br />
                left click    pin D3      <br />
                


Rotary encoder:               <br />

  scroll up     pin D6        <br />
  scroll down   pin D7        <br />
  right click   pin D4        <br />
